## Jogo da Velha 
Projeto da disciplina "Programação Estruturada" do curso de Ciência de Ddaos e IA (UFPB).

### Objetivo
Construir um Jogo da Velha na linguagem C, onde o usuário possa escolher entre 3
opções de jogo:
- Jogo entre dois adversários humanos
- Jogo entre um adversário humano e o computador
- Jogo entre um humano ou o computador e um jogador remoto
